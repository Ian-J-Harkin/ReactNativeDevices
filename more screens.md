# Additional Identified Screens from Axicube Portfolio Images

Based on the images found in the `SportPlus Fitness App _ Axicube_files` directory (specifically the large and small presentation images), the following app screens can be identified:

1. **Welcome / Onboarding Screen**
   - Shows inspirational imagery and "Shape your perfect body" text, with a "Get started" button.
   - Another variation shows "Create account" and "Log in" buttons.

2. **Dashboard / Home Screen**
   - User greeting ("Hello, John")
   - Weekly overview summary (total workouts, minutes active, calories burned)
   - "Start previous workout" quick-action button
   - List of recent "Previous Workouts"
   - Bottom Tab Navigation (Dashboard, Workout, Quick Workout)

3. **Profile & Account Switcher Screen**
   - User avatar and basic stats (Sex, Weight, Height, Age)
   - A bottom sheet modal to switch accounts or "Add new account"

4. **My Devices / Bluetooth Search**
   - Radar UI showing "Search of devices nearby"
   - Tells the user to turn on Bluetooth to connect the app

5. **Select / Add Device Modal**
   - List of found or supported devices (X-Bike, Crosstrainer) with connection statuses
   - "Let's go" button to confirm selection

6. **Select Workout Screen (Browse)**
   - Top search bar 
   - Filter chips for equipment types (All, X-Bike, Crosstrainer, Rudergerät)
   - Scrollable list of workout cards showing an image, title, level, and duration

7. **Workout Details Screen**
   - Specific workout overview (e.g., "X-Bike daily workout")
   - Confirmation if the necessary device is "CONNECTED"
   - Workout description and chart of previous workout stats
   - "Start workout" button

8. **Workout Setup Screen**
   - Pre-workout configuration allowing the user to set duration (15 min, 30 min, 45 min, Custom)
   - Initial resistance level selector

9. **Active Workout Screen (Dark Mode)**
   - Live dashboard during exercise showing elapsed time, resistance level, watts, bpm, distance, and calories burned
   - Large pause/play toggle, typically rendered in a high-contrast dark theme

10. **Workout Summary / Congratulations Screen**
    - "Congratulations! You finished current workout"
    - Final tallies for the session (duration, distance, calories, avg bpm, etc.)
    - "Share stats" button

11. **Workout History & Detailed Stats Screen**
    - Weekly aggregate overviews (e.g., "18 April - 25 April")
    - Breakdown of specific workout sessions with bar charts detailing performance throughout the session
    - "Share" functionality
